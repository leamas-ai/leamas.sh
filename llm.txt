## Project Overview
Leamas is an open source package manager for Claude AI agents and commands. It provides a simple CLI tool to install, manage, and distribute AI agent kits and command extensions for Claude AI.

## What is Leamas?
- **Package Manager**: Install Claude AI agent kits and commands with simple commands
- **Open Source**: Fully open source project welcoming contributions
- **Easy Installation**: One-line installer script for quick setup
- **Kit Registry**: Centralized repository of community-contributed AI agents and commands
- **Cross-Platform**: Works on macOS, Linux, and Windows
- **Community-Driven**: Built by and for the Claude AI community

## Key Features
- Simple CLI interface: `leamas agent@name` or `leamas command@name`
- Global and local installation options
- Automatic dependency checking
- Built-in update mechanism
- Kit caching for faster installs
- Comprehensive help and documentation

## Installation
```bash
# Quick install
curl -sSL https://raw.githubusercontent.com/leamas-ai/leamas.sh/main/install.sh | bash

# Manual install
wget https://raw.githubusercontent.com/leamas-ai/leamas.sh/main/leamas
chmod +x leamas
./leamas --help
```

## Usage Examples
```bash
# List available kits
leamas --list

# Install an agent kit
leamas agent@developer-assistant

# Install a command kit
leamas command@code-review

# Install globally
leamas -g agent@project-manager

# Check version
leamas --version

# Update leamas
leamas --update
```

## Technical Details
- **Language**: Bash shell script
- **Dependencies**: curl, standard Unix tools
- **License**: Open source (likely MIT or Apache 2.0)
- **Repository**: https://github.com/leamas-ai/leamas.sh
- **Install Location**: `$HOME/leamas/` (default)
- **Global Install**: `$HOME/.claude/agents/leamas/` or `$HOME/.claude/commands/leamas/`
